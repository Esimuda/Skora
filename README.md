# Skora RMS - Nigerian School Result Management System

Complete offline-capable result management system for Nigerian secondary schools.

## Features
- ✅ Complete offline support
- ✅ Nigerian grading system (CA1 + CA2 + Exam)
- ✅ Automatic position calculation
- ✅ Behavioral assessment (8 metrics)
- ✅ Multi-class management
- ✅ PDF generation with school branding
- ✅ All data in UPPERCASE

## Quick Start

```bash
npm install
npm run dev
```
